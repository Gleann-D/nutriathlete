Plan nutritionnel post-séance
📱
Configurer le Raccourci iOS
HealthFit → NutriAthlète · Zéro intermédiaire · 3 étapes
1
Créer le Raccourci
Ouvre l'app Raccourcis sur iPhone → + → nommer le raccourci "Analyse Séance".
2
Ajouter les 3 actions
Dans l'éditeur, ajouter ces actions dans l'ordre :
// Action 1 — Récupère le dernier workout Apple Health
Santé → Chercher des données de santé
Type : Exercice physique
Limite : 1  ·  Tri : Date de début (récent en premier)

// Action 2 — Récupère la FC moyenne récente
Santé → Chercher des données de santé
Type : Fréquence cardiaque
Limite : 1  ·  Tri : Date (récent en premier)

// Action 3 — Ouvre l'app avec les données en URL
Safari → Ouvrir les URL
→ coller l'URL ci-dessous, remplacer les [variables]
URL à coller dans l'action "Ouvrir les URL" — les [crochets] sont des variables dynamiques à insérer depuis les résultats des actions 1 & 2 :
https://TON-URL.com
?discipline=[Type d'exercice du workout]
&duration=[Durée du workout en minutes]
&distance=[Distance du workout en km]
&calories=[Calories actives du workout]
&hr_avg=[Valeur de la FC]
&start=[Heure de début du workout]
&autolaunch=1
💡 Dans Raccourcis, tape dans le champ URL et sélectionne "Insérer variable" pour chaque [crochet]. Le paramètre autolaunch=1 déclenche l'analyse immédiatement sans clic supplémentaire.
3
Automatiser avec une Automation
Dans Raccourcis → onglet Automatisation → + → Personnelle :
Déclencheur : Chargeur connecté
Heure : Entre 19h00 et 23h30
Action : Exécuter "Analyse Séance"
Confirmation : Désactivée (exécution immédiate)
🎯 Flux complet : Tu rentres de ta séance → tu branches ton iPhone → l'app s'ouvre automatiquement avec ta séance déjà remplie → l'IA génère ton plan nutrition en quelques secondes.
